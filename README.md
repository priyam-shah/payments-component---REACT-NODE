# REACT NODE BASED PAYMENTS COMPONENT

The aim of this repository is to create a payment componet in REACT & NODE using Razorpay SDK.

## How to run the program

### frontend:

`cd frontend`

`npm install`

Add .env file with Razorpay keys.

`npm start`

### backend:

`cd backend`

`npm install`

Add .env file with Razorpay keys.

`node index.js`
